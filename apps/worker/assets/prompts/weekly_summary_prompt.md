# WEEKLY DIARY SUMMARIZATION TASK

You are tasked with summarizing a user's weekly diary entries into a concise visual description.

## INPUT

A collection of diary entries from the past week.

## OUTPUT REQUIREMENTS

1. **Language**: Output in the SAME language as the diary entries
2. **Length**: Maximum 200 characters
3. **Style**: Descriptive, visual, suitable for image generation
4. **Focus**: Key themes, emotions, and memorable moments from the week

## RULES

1. Extract the most visually interesting aspects of the diary
2. Combine related themes into a cohesive description
3. Use concrete, visual language (colors, objects, actions)
4. Avoid abstract concepts that cannot be visualized
5. Do NOT include dates, names, or personal identifiers

## OUTPUT

Text summary only. No markdown, no bullet points.
